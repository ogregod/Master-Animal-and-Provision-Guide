## II. § NOTATION GUIDE

### **HOW TO READ CROSS-REFERENCES**

Throughout this guide, you'll see references like:
- **§02_WHITETAIL** = See file "02_SWORD_COAST_WHITETAIL.md"
- **§01_UNIVERSAL_CERVINE_CUTS** = See file "01_UNIVERSAL_CERVINE_CUTS.md"
- **§APPENDIX_A** = See file "APPENDIX_A_BREED_COMPARISON.md"

**§ Symbol = Cross-reference marker** (directs you to another file)

---

### **FILE NAMING CONVENTIONS**

**Foundation files:**
- **00_** = Front matter, introductory material
- **01_** = Universal information (applies to all breeds)

**Breed files:**
- **02_** = Common deer (Part 2: 5 breeds)
- **03_** = Large game (Part 3: 4 breeds)
- **04_** = Exotic (Part 4: 2 breeds)
- **05_** = Dangerous trophy (Part 5: 1 breed)

**Appendices:**
- **APPENDIX_A** = Breed comparison tables
- **APPENDIX_B** = Cuts and prices reference
- **APPENDIX_C** = Antler trophy mastery
- **APPENDIX_D** = Foundation cross-reference (this file)

---

### **EXAMPLE CROSS-REFERENCE USAGE**

**In breed file, you might see:**
> "For universal field dressing procedures, see §01_UNIVERSAL_CERVINE_CUTS, pages 5-10."

**This means:**
1. Open file "01_UNIVERSAL_CERVINE_CUTS.md"

## V. FORMULA REFERENCE SHEET

### **UNIVERSAL YIELD FORMULA (ALL BREEDS)**

**LIVE TO RETAIL YIELD:**
```
Live Weight × 65% = Dressed Weight (hide, head, organs removed)
Dressed Weight × 45% = Retail Weight (bone, trim removed)

COMBINED:
Live Weight × 29% = Retail Weight
```

**EXAMPLE (Whitetail Deer):**
```
180 lbs live × 65% = 117 lbs dressed
117 lbs dressed × 45% = 53 lbs retail

OR:
180 lbs live × 29% = 52 lbs retail (rounding)
```

**This formula works for ALL cervines** (60 lb Roe Deer to 2,000 lb Giant Elk)

### **ANTLER SCORING FORMULA (GUILD OF HUNTERS)**

**400 POINT MAXIMUM:**
```
Spread (inches) × 1.6 = Points (160 max)
Main Beam Length (inches) × 1.25 = Points (100 max)
Total Point Count × 2 = Points (80 max)
Average Circumference (inches) × 5 = Points (40 max)
Symmetry = Subjective (20 max)

TOTAL = Sum of all components (max 400)
```

**Trophy Classes:**
- Common: <240 points
- Good: 240-300 points
- Trophy: 300-360 points
- Record: 360+ points
